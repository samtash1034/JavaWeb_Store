# 簡介
基於servlet+jsp+mysql做的一個簡易的網上書店項目源碼

# 使用技術
Jsp

Servlet

MySQL

# 功能介紹

### 1、登入功能
* 登入成功則跳轉到首頁
* 登入失敗跳轉到登入頁面並顯示提示訊息

* 注意生日格式不可打錯
* 註冊成功跳轉到登入頁面
* 註冊失敗跳轉到註冊頁面並顯示登入訊息
